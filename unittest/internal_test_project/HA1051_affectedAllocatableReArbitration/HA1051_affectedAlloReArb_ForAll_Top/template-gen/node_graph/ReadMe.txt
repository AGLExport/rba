NodeGraph

■概要
RBAモデルの要素(エリア、ゾーン、表示コンテント、音声コンテント、制約、後処理制約、制御実行式、シーン)と要素間の関係を可視化するグラフを生成します。

■使い方
Graphvizを利用します。
Graphviz(https://www.graphviz.org/download/)をインストールして、
環境変数PATHにdot.exeへのパス(e.g. "C:\Program Files (x86)\Graphviz2.38\bin")を設定してください。

■グラフの説明
各モデル要素はノードとして表示されます
・表示コンテント、音声コンテント
　楕円(枠線：黒、塗りつぶし：なし)
・表示コンテント集合、音声コンテント集合
　楕円(枠線：なし、塗りつぶし：緑)
・エリア、ゾーン
　長方形(枠線：黒、塗りつぶし：なし)
・エリア集合、ゾーン集合
　長方形(枠線：なし、塗りつぶし：ピンク)
・制約
 右上の欠けた長方形(枠線：なし、塗りつぶし：グレー)
・後処理制約
 右上の欠けた長方形(枠線：なし、塗りつぶし：ラベンダー)
・制御実行式
 右上の欠けた長方形(枠線：なし、塗りつぶし：薄いピンク)
・シーン
　円(枠線：なし、塗りつぶし：水色)

要素間の関係
・コンテント-エリア/ゾーン
　矢印なしの赤線
・コンテント集合→コンテント、エリア集合/ゾーン集合→エリア/ゾーン
　矢印ありの黒線
・制約/後処理制約/制御実行式→その他の要素
　矢印ありの青線
